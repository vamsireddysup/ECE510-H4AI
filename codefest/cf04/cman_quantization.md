# CMAN -- INT8 symmetric quantization

**Weight matrix W (FP32):**
```
W = [  0.85, -1.20,  0.34,  2.10 ]
    [ -0.07,  0.91, -1.88,  0.12 ]
    [  1.55,  0.03, -0.44, -2.31 ]
    [ -0.18,  1.03,  0.77,  0.55 ]
```

---

## Task 1: Scale factor

Max absolute value in W = **2.31** (element W[2][3])

```
S = max(|W|) / 127 = 2.31 / 127 = 0.01819
```

---

## Task 2: Quantize W_q = round(W / S), clamped to [-128, 127]

```

[  -9.90,  56.62,  42.33,  30.24 ]
```

No clamping required -- all values within [-128, 127].

```

[ 0.0038, 0.0064, 0.0034, 0.0001 ]
```

```
MAE = (0.0049+0.0005+0.0056+0.0081+0.0028+0.0005+0.0064+0.0073
      +0.0038+0.0064+0.0034+0.0001+0.0019+0.0068+0.0060+0.0043) / 16
    = 0.0688 / 16

```

## Task 5: Bad scale experiment (S_bad = 0.01)

```
W / S_bad:
[  85, -120,   34,  210* ]
[  -7,   91, -188*,  12  ]
[ 155*,   3,  -44, -231* ]
[ -18,  103,   77,   55  ]
* = clamped
```

**W_q_bad (INT8, after clamping):**
```
[  85, -120,   34,  127 ]
[  -7,   91, -128,   12 ]
[ 127,    3,  -44, -128 ]
[ -18,  103,   77,   55 ]
```

**W_deq_bad = W_q_bad x 0.01:**
```
[  0.85, -1.20,  0.34,  1.27 ]
[ -0.07,  0.91, -1.28,  0.12 ]
[  1.27,  0.03, -0.44, -1.28 ]
[ -0.18,  1.03,  0.77,  0.55 ]
```

**Error |W - W_deq_bad|:**
```
[ 0.00, 0.00, 0.00, 0.83 ]
[ 0.00, 0.00, 0.60, 0.00 ]
[ 0.28, 0.00, 0.00, 1.03 ]
[ 0.00, 0.00, 0.00, 0.00 ]
```

```
MAE_bad = (0.83 + 0.60 + 0.28 + 1.03) / 16 = 2.74 / 16 = 0.171
```

MAE increased from **0.0043** to **0.171** -- 40x worse.

When S is too small, large values in W exceed the INT8 range after division
and get clamped to ±127/128, permanently losing that information and
producing large dequantization errors.
